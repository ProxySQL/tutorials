# ProxySQL Tutorials that use Docker

Docker is an open-source platform for running application in containers.

To use the **ProxySQL** tutorials that use Docker you will require:

1. Docker
2. Docker Compose
3. Git

- [Docker Deskop](https://docs.docker.com/desktop/)
  - [Mac OS](https://docs.docker.com/desktop/setup/install/mac-install/)
  - [Windows](https://docs.docker.com/desktop/setup/install/windows-install/)
  - [Linux](https://docs.docker.com/desktop/setup/install/linux/)
- Docker Compose is included with Docker Desktop

Several other products offer the same functionality and can also be used. These include:
- [Rancher Desktop](https://rancherdesktop.io/) - Supports Mac OS (Silcon & Intel), Windows (x64) and Linux
- [Podman](https://podman.io/) - Supports Mac OS, Windows and Linux
- [Colima](https://colima.run/) - Support Mac OS and Linux
