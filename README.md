# ProxySQL Tutorials

These tutorials demonstrate various features of [ProxySQL](https://proxysql.com).

<b>ProxySQL</b> is a high‑performance, high‑availability transparent proxy for PostgreSQL, MySQL and
PostgreSQL and MySQL wire-compatible products, and cloud-managed implementations.

- **WebSite**: [proxysql.com](https://proxysql.com)
- **GitHub**: https://github.com/sysown/proxysql

We welcome community discussions and contributions, including with [**Group Groups**](https://groups.google.com/g/proxysql) and GitHub [**Issues**](https://github.com/sysown/proxysql/issues).  
To keep up with the latest information, check out our [**Blog**](https://proxysql.com/blog/), follow us on [**LinkedIn**](https://www.linkedin.com/company/proxysql/), or subscribe to our [**Newsletter**](https://proxysql.com/community/).

These tutorials are broken down into the following categories:
- MySQL tutorials in `mysql`
- PostgreSQL tutorials in `postgresql`

Tutorials will specify if they can be run locally via `docker` or require a specific cloud provider or other environment setup.

## Getting Started

Simply clone this repository, switch to a tutorial and read the instructions.
```
git clone https://github.com/ProxySQL/tutorials.git
```
