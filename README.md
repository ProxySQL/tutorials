# ProxySQL Tutorials

These tutorials and examples demonstrate [ProxySQL](https://proxysql.com) functionality.

ProxySQL is a high‑performance, high‑availability transparent proxy for PostgreSQL, MySQL and
PostgreSQL and MySQL wire-compatible products.

- **Website**: [proxysql.com](https://proxysql.com)
- **GitHub**: https://github.com/sysown/proxysql

We welcome discussions and contributions, including with [**Group Groups**](https://groups.google.com/g/proxysql) and GitHub [**Issues**](https://github.com/sysown/proxysql/issues).  
To keep up with the latest information, check out our [**Blog**](https://proxysql.com/blog/), follow us on [**LinkedIn**](https://www.linkedin.com/company/proxysql/), or subscribe to our [**Newsletter**](https://proxysql.com/community/).

These tutorials are broken down into:
- `mysql` tutorials
- `postgresql` tutorials

Tutorials will specify if they can be run locally via `docker` or require a specific cloud provider or other environment setup.

## Getting Started
```
git clone https://github.com/ProxySQL/tutorials.git
```
